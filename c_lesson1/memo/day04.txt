	2. 관계(비교)연산자
		a > b	: a가 b보다 크니? -> 맞으면 1, 아니면 0
	
		a < b	: a가 b보다 작니?
		
		a >= b 	: a가 b보다 크거나 b와 같니?
		
		a <= b	: a가 b보다 작거나 b와 같니?

		a == b	: a와 b가 같니?
	
		a != b	: a와 b가 다르니? -> 맞으면 1, 아니면 0

	3. 논리연산자
		a && b	: 논리곱, and 연산
				(a가 참이고, b도 참이면 결과는 참, 아니면 거짓)
				a가 0이 아니고, b도 0이 아니면 결과는 1, 아니면 0
		
		a || b	: 논리합, or 연산
				(a가 참 또는 b가 참이면 결과는 참, 아니면 거짓)
				a 또는 b 둘 중 하나라도 0이 아니면 결과는 1, 아니면 0

		!a	: 논리부정, not 연산
				(a가 참이라면 결과는 거짓, a가 거짓이라면 결과는 참)
				a가 1이라면 결과는 0, a가 0이라면 결과는 1
		
		int num = 55;
		num > 50 && num < 100;

	4. 삼항연산자 ★
		조건식 ? 참일 때의 값 : 거짓일 때의 값;
		
		조건식이 참이면: 앞쪽 값이 연산의 결과로 나타난다.
		조건식이 거짓이면: 뒤쪽 값이 연산의 결과로 나타난다.
		참일때 나올 값과 거짓일때 나올 값의 자료형은 통일되어 있어야 한다.

※ 조건식
: 관계 연산자 혹은 논리 연산자를 사용하여 0 혹은 0이 아닌 숫자(참 또는 거짓)로 판별할 수 있는 식
