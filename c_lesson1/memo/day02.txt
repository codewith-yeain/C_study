""의 역할
	명령어가 아닌 문자열임을 구분할 수 있게 하기 위해 사용한다.

값의 종류(자료형 type)
	char	문자	'a', 'A'...	1byte
	int	정수	10, 2, -5...	4byte
	double	실수	10.2, -3.1 ...	8byte

변수
	값을 저장하는 '저장공간'
	값을 재사용 하기 위해서 사용한다.

변수의 선언
	자료형 변수명;
	ex) int num;

변수의 초기화
	변수명 = 값;
	
	선언된 변수에 최초의 값을 대입하는 행위
	선언된 변수 타입에 알맞은 값만 대입해 주어야 한다.

변수의 선언과 초기화 동시에
	자료형 변수명 = 값;

변수와 값 구분하기
	변수 이름 오른쪽에 대입 연산자가 있다면 저장공간으로 쓰인다.
	대입연산자가 없다면 ★★★★값★★★★으로 쓰인다!

파일 빌드에서 제외하는 방법
	해당 파일 마우스 오른쪽 버튼 클릭
	> 속성 > 일반 > 빌드에서 제외 > 예 > 확인

제어문자
	문자열 안에서 특수한 기능을 하는 데에 사용되는 문자
	
	\n: 엔터를 표현한다.
	\t: tab을 표현한다.
	\": 쌍따옴표를 표현한다.
	\\: 역슬래시를 표현한다.

형식지정자
	char	%c
	int	%d
	double	%lf
	주소값	%p

%[-][n][.m] 형식 지정자
	ex) %-10.2lf
	
	[-]: 왼쪽 정렬을 시켜준다.(%d나 %c에서도 사용 가능)
	[n]: 전체 자릿수를 n으로 설정한다.(%d나 %c에서도 사용 가능)
	[.m]: 소숫점 아래 자릿수를 m으로 설정한다.(%lf에서만 사용 가능)
