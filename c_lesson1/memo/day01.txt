C언어
	1. 절차지향 언어이다.
		: 정해진 순서에 따라 실행된다는 의미

	2. C 언어는 저급언어이다.
		: 저급 언어라는 것은 프로그램의 기반을 이루는 언어(컴퓨터 친화적)
		

컴파일
	사람의 언어(명령어)를 컴퓨터 언어(기계어)로 번역하는 행위


visual studio 2019 16.11 버전 다운로드 방법
	구글 -> visual studio 2019 검색 -> 최상단 링크 클릭 -> 2019 다운로드 클릭
	-> 로그인 -> Visual Studio Community 2019 (version 16.11) 다운로드 클릭
	-> 다운로드 완료 후 잘라내기 -> 우리 폴더 resource로 붙여넣기 -> 실행
	-> 'c++ 을 이용한 데스크탑 개발' 만 체크 -> 설치 시작

기본 구성 요소
	솔루션
		프로젝트
			소스파일
				메인함수
					소스코드

함수
	이름 뒤에 소괄호
	어떠한 기능을 한다
	printf() : 괄호 안의 값을 콘솔에 출력해주는 기능

콘솔창
	개발자와 컴퓨터가 소통한 결과를 보여주는 창

main() 함수
	ctrl + F5 를 눌러 프로그램을 실행시켰을 때
	메인 함수 안쪽의 코드를 순차적으로 컴파일하는 기능을 한다.
	(프로그램 실행 == 메인함수 안쪽코드 실행)

새 프로젝트 만들기 > 빈 프로젝트 > 프로젝트 이름 basic > 솔루션 및 프로젝트를 같은 디렉터리에 배치 체크
> 위치를 workspace로 설정 > 만들기 클릭 > 소스파일 우클릭 + 추가 + 새항목 > C++파일 선택 > 이름 basic.c
> 추가 선택 >  int main() {} 작성 -> 헤더파일 소속 알려주기 #include <stdio.h> 작성
